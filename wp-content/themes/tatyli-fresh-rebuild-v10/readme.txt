A clean WordPress theme for Tatyli, a non-profit cultural organization based in Liège, Belgium.

== Notes ==
The theme includes original vector logo and hero artwork, starter pages, responsive styles, and a simple contact form handler.


V4 fixes:
- Header is no longer sticky, so it does not cover page content during scrolling or full-page screenshots.
- Starter pages are repaired automatically after theme updates, including Privacy Policy.
- Footer address is shown on one line.
- Home page section markup corrected.

V6 fixes:
- Privacy Policy simplified to match the current website contact form and basic technical operation.


V7 checks/fixes:
- Contact form now sends directly to info@tatyli.be instead of relying on the WordPress admin email.
- Added a hidden honeypot field to reduce simple contact form spam.
- Added a visible mail-error message if WordPress cannot send the message.

V8/V9 privacy fixes:
- Removed the long complaint section and broad service-use statement from the Privacy Policy.
- Simplified legal basis wording for a contact-form-only website.
- Removed theme-specific wording from the public Privacy Policy.

V10 final review fixes:
- Version metadata is consistent across theme files.
- Added a fallback SVG favicon link when no WordPress Site Icon is set.
